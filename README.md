# Kanban made with Vue3

> This project is a kanban todo app made with vue 3 for learning purposes

**Dependencies**

- NodeJS 12+
- Vite
- Vue3
- Vue router
- Vuex 4

## Running

- Clone this repo

```
git clone https://github.com/rbentivenha/kanban-web.git
yarn dev OR
yarn build
```
